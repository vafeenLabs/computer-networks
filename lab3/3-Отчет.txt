Создал 10 локальных подсетей
При помощи протокола RIP получил доступ из каждой подсети в кажду путём
прописывания всех подсетей в нужном окне

Для того что бы соеденить роутеры при помощи протокола rip, нужно в каждом
роутере в прописать все соседние подсети.

Для того что бы посмотреть, что роутеры соеденины именно при помощи Rip протокола,
нужно открыть таблицу маршрутизации
R в столбце type означает rip, можно обратить внимание на метрику, она отличается от connected и static.

Настроил DHCP сервер для всех устройств во всех локальных подсетях 
при помощи 2-х команд:
interface <Название интерфейса>
ip helper-address <ip 	DHCP сервера>
Настроил HTTP сервер для всех устройств во всех локальных подсетях
Настроил DNS сервер для всех устройств во всех локальных подсетях
Настроил почтовый сервер для всех устройств во всех локальных подсетях

Альтернативные маршруты тоже работают. Пример:
Для попадания из сети 192.168.1.0 в 192.168.2.0 пакет пойдёт через сеть 192.168.12.0

Если оторвать этот провод, то пакет полетит сначала через 192.168.13.0, а затем
в 192.168.23.0, после чего попадёт в нужную сеть
